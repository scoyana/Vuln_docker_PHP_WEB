Frome KnockOn 
🛠 게시판 만들기 프로젝트 명세서
📌 과제: LAMP 웹 서버 기반 게시판 구축
🎯 목표
LAMP(Linux + Apache + MySQL + PHP) 웹 서버의 문법, 구조, 동작 방식, 사용 방법 이해
웹 해킹 실습을 위한 실습용 서버 및 애플리케이션 환경 구축
✅ 세부 명세서
1️⃣ 사용 환경 및 기술 스택
운영체제: Linux (Ubuntu 20.04 이상 권장)
웹 서버: Apache2
DBMS: MySQL
서버 언어: PHP (7.4 이상 권장)
기타: HTML/CSS (기본 UI 구현용), JavaScript(Optional), Docker(Advanced)
2️⃣ 필수 기능 목록
구분	기능 설명
📄 게시물 목록	여러 개의 게시물을 리스트 형태로 보여주는 메인 화면 구현 (최신순 정렬)
🔍 게시물 검색	제목 또는 내용 기준으로 키워드 검색 기능 구현
➕ 게시물 작성	제목, 내용, 파일 업로드(첨부파일)를 포함한 글 작성 기능
✏️ 게시물 수정	본인이 작성한 글만 수정 가능하게 구현 (세션 기반 검증)
❌ 게시물 삭제	본인이 작성한 글만 삭제 가능하게 구현 (세션 기반 검증)
📎 파일 업로드	게시글 당 첨부파일 1개 이상 가능 (서버에 저장하고 다운로드 링크 제공)
👤 회원가입	사용자 ID/PW 등록 기능 구현, 비밀번호는 단방향 해시(SHA256, bcrypt 등) 처리 권장
🔐 로그인/로그아웃	쿠키/세션 기반 로그인 상태 유지 기능 구현
